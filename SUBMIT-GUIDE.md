## 🔧 Google Search Console 配置（解决索引问题）

你的网站显示 "已发现 - 尚未编入索引"，说明已经发现了但还没抓取。

**需要做的:**
1. 登录 [Google Search Console](https://search.google.com/search-console)
2. 选择 `aisticker.pics` 资源
3. 点击 "抓取" → "请求建立索引"
4. 提交 sitemap: `https://aisticker.pics/sitemap.xml`
5. 在 "设置" → "抓取频率" 中保持默认

**为什么只收录了主页?**
- 博客页面是动态生成的，Google 需要时间发现
- 提交 sitemap 后，Googlebot 会逐步抓取所有页面
- 通常需要 1-4 周时间

**加速收录的方法:**
- 在这些 AI 工具目录提交后（有外链），Google 会更快抓取
- 可以在 Search Console 中手动请求收录每个博客页面
- 确保 sitemap.xml 包含所有博客 URL
